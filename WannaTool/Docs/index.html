WannaTool

The ultimate power user launcher for Windows

Web search & redirects

  • google: query Google search
  • bing: query Bing search
  • ddg: query DuckDuckGo
  • brave: query Brave search
  • youtube: query YouTube
  • wiki: query Wikipedia
  • stackoverflow: query Stack overflow

File filters

  • exe: name Apps (.exe, .lnk)
  • pdf: name PDF documents
  • doc: name Word / Text docs
  • img: name Images
  • code: name Source code

System & screenshots

  • Ctrl + Shift + F5 Capture primary screen
  • Ctrl + Shift + F6 Capture all screens
  • !help Open help
  • !settings Open settings
  • !exit Exit WannaTool
Screenshots saved to:
Pictures/WannaTool_Captures/

Process control

  • !top List top heavy processes
  • !kill Select process to terminate
  • !kill name Kill specific process
