WannaTool - Command Reference

Your ultimate command launcher for Windows

Search & Redirects

  • google: query – Search Google
  • bing: query – Search Bing
  • !help – Open this help page

System Tools

  • CTRL+SHIFT+F5 – Screenshot primary screen
  • CTRL+SHIFT+F6 – Screenshot all screens
  • Screenshots are saved in Images/WannaTool_Captures/

Local File Features

  • pdf: name – Quickly open a PDF from your files
  • img: name – Find and open images
  • code: name – Locate source files
